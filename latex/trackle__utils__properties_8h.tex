\doxysubsection{Detailed Description}
Datatypes and functions for working with properties. Read the full description to learn about properties. 

There are two concepts when it comes to working with properties\+:
\begin{DoxyItemize}
\item Properties;
\item Properties groups.
\end{DoxyItemize}

Properties are variables that are periodically published to the cloud with their name (key).

Properties groups are groups of properties that are published with the same time period.

Only properties that are inside a group are published to the cloud.

Properties groups are created as follows\+:
\begin{DoxyEnumerate}
\item Declare a variable of type \mbox{\hyperlink{trackle__utils__properties_8h_a25827b24834b1715c9273f8284320480}{Trackle\+\_\+\+Prop\+Group\+ID\+\_\+t}};
\item Assign the result of \mbox{\hyperlink{trackle__utils__properties_8h_aa9d1943c7c82f5c2e8c986d5eead2dc7}{Trackle\+\_\+\+Prop\+Group\+\_\+create}} to this variable;
\item Repeat the steps from 1 to 2 for all the properties groups that must be created;
\end{DoxyEnumerate}

Then, in order to create properties, one must follow these steps\+:
\begin{DoxyEnumerate}
\item Declare a variable of type \mbox{\hyperlink{trackle__utils__properties_8h_a44a453fbe529a53629bcf4d40c8595de}{Trackle\+\_\+\+Prop\+ID\+\_\+t}};
\item Assign the result of \mbox{\hyperlink{trackle__utils__properties_8h_a12988fbba8c3a1d91a11995df93a1a88}{Trackle\+\_\+\+Prop\+\_\+create}} to this variable;
\item Add the created property to one or more groups with \mbox{\hyperlink{trackle__utils__properties_8h_a112541983423fb247ab6f6a05900863a}{Trackle\+\_\+\+Prop\+Group\+\_\+add\+Prop}};
\item Repeat the steps from 1 to 3 for all the properties that must be created;
\item Call \mbox{\hyperlink{trackle__utils__properties_8h_a12db7e4d66706a3c4053c73ca2812071}{Trackle\+\_\+\+Props\+\_\+start\+Task}} to start the properties task.
\end{DoxyEnumerate}

Now, one can work with properties (update, read, etc.) by using the remaining functions exposed by this file.
